# 动漫角色识别网站

这是一个基于 dio.jite.me API 的动漫角色识别网站，可以上传图片并识别其中的动漫角色。

## 功能特性

- 上传图片进行动漫角色识别
- 显示识别结果（角色名称、作品、置信度等）
- 支持全局修正功能
- 最多支持识别400个人脸

## 技术栈

- 前端：HTML, CSS, JavaScript
- 后端：Node.js, Express
- API：dio.jite.me 识别API

## 安装与运行

1. 安装依赖：
   ```bash
   npm install
   ```

2. 启动服务：
   ```bash
   npm start
   ```

3. 访问网站：
   打开浏览器访问 http://localhost:3000

## 使用方法

1. 在网站上选择一张包含动漫角色的图片
3. 等待识别结果
4. 查看识别出的角色信息
